# EDD Mixpanel Lite

EDD Mixpanel Lite makes it easy to connect Mixpanel with Easy Digital Downloads.

## Lite Features

* Customer Profiles
* Added to Cart
* Completed Purchase

## Premium Features (Coming soon!)

* Signed In
* Signed Out
* Viewed Signup
* Signed Up
* Viewed Homepage
* Removed from Cart
* Viewed Cart
* Applied Coupon
* Cancelled Order
* and more!
